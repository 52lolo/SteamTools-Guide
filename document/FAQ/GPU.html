# 软件使用GPU

Watt Toolkit 使用显卡进行界面硬件渲染加速

在打开主程序界面的时候会调用 GPU 渲染

如果不想要占用 GPU 可以在设置中关闭

设置 -> 通用设置 -> 关闭 使用GPU渲染界面(硬件加速)
