# 本地开发

先决条件：

  • nodejs(Version >= 16)
  • yarn(Version >= 1)

Clone 本仓库

在项目文件夹下运行：

yarn install
yarn docs:dev

更新框架及插件(非必要不必更新)

yarn add -D vuepress@next
yarn add @vuepress/plugin-search@next -d
